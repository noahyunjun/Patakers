# 02. Content IA

## Narrative Flow
1. Hero: 정체성 선언
2. Why Worship: 왜 찬양하는가
3. Worship History: 실제 예배 기록
4. Message Archive: 카드뉴스/포스터 무드
5. Join Us: 함께하기 CTA

## Section Objective + CTA
- Hero: 팀 핵심 가치 전달 / `예배 이야기 보기`
- Why Worship: 신학적/공동체적 이유 제시 / `아래로 스크롤`
- Worship History: 신뢰 형성 / `다음 모임 확인`
- Archive: 감성/브랜드 일관성 증명 / `인스타 연동(후속)`
- Join Us: 행동 유도 / `함께하기 문의 보내기`

## Copy Rules
- 헤드라인: 8~14자 중심
- 본문: 2~3문장 제한
- 강조 단어: 섹션당 1개
