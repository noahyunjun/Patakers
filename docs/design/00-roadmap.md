# Partakers Design Execution Roadmap

## Goal
디자이너 퀄리티의 일관된 랜딩 페이지를 만들기 위한 실행 프레임입니다.

## 8-Step Execution
1. Brand System 확정 (`01-brand-system.md`)
2. IA/콘텐츠 구조 확정 (`02-content-ia.md`)
3. 레퍼런스 해체 분석 (`03-reference-deconstruction.md`)
4. 시각 계층 규칙 고정 (`04-visual-hierarchy.md`)
5. 그리드/레이아웃 리듬 고정 (`05-grid-rhythm.md`)
6. 모션 시스템 고정 (`06-motion-system.md`)
7. 디자인 QA 루프 (`07-design-qa-checklist.md`)
8. Figma -> 구현 핸드오프 (`08-figma-workflow.md`)

## Working Rules
- 한 번에 한 단계씩 잠그고 다음 단계로 이동
- 잠긴 단계는 변경 시 변경 로그 남김
- 구현 전에는 반드시 문서 체크리스트 통과
