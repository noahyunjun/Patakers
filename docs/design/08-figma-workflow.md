# 08. Figma Workflow

## File Structure
- Page 1: Foundations (color/type/spacing/motion)
- Page 2: Wireframe (low fidelity)
- Page 3: UI (high fidelity)
- Page 4: Prototype (scroll flow)
- Page 5: Handoff (component spec)

## Handoff Contract
- 각 컴포넌트에 다음 정보 포함:
- size, spacing, states(default/hover/focus/disabled)
- typography token name
- color token name
- motion token name

## Dev Sync Process
1. Figma 승인
2. 코드 반영
3. QA 체크리스트 실행
4. 스냅샷 리뷰
