# 03. Reference Deconstruction

## Mason Wong Style (Applied)
- 큰 타이포 중심 Hero
- 과하지 않은 스크롤 기반 집중 전환
- 여백 중심 편집 디자인
- 강한 한 문장 + 절제된 보조문

## Partakers Adaptation
- 포트폴리오 감성 -> 말씀/공동체 서사로 변환
- 어두운 Hero + 저채도 카드 조합
- 빈티지 텍스처 오버레이 사용
- 과장된 인터랙션 배제

## Reuse Matrix
- Keep: 타이포 임팩트, 여백, 스크롤 흐름
- Modify: 카피, 컬러, 신앙 맥락
- Avoid: 과도한 실험적 인터랙션
