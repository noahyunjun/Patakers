# 07. Design QA Checklist

## Layout
- [ ] 375px, 768px, 1440px에서 깨짐 없음
- [ ] Hero 텍스트가 fold 안에 들어옴
- [ ] CTA 버튼이 터치 가능 크기(최소 44px)

## Typography
- [ ] H1/H2 계층이 명확
- [ ] 본문 line-length 과도하지 않음
- [ ] 한 섹션 내 폰트 스타일 난립 없음

## Color/Contrast
- [ ] 본문 대비 4.5:1 이상
- [ ] 강조색 남용 없음

## Motion
- [ ] 스크롤 시 깜빡임 없음
- [ ] reduce-motion에서 정적 동작

## Content
- [ ] 섹션 목적과 CTA가 일치
- [ ] 문구 톤이 일관됨
