# 04. Visual Hierarchy Rules

## Type Scale
- Hero H1: `clamp(2.4rem, 8vw, 5.8rem)`
- Section H2: `clamp(1.8rem, 4vw, 3rem)`
- Card H3: `1.4rem ~ 1.5rem`
- Body: `1rem ~ 1.15rem`
- Eyebrow: `0.74rem`, uppercase, wide letter spacing

## Emphasis Rules
- 섹션별 강조 포인트 1개만 사용
- 밝기 대비는 최소 4.5:1 유지
- 버튼 스타일은 Primary/Secondary 2종만 사용

## Copy Hierarchy
- 한 화면에서 메시지 우선순위는 최대 3단계만 허용
